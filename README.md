# MLOps Signal Generation Pipeline

A miniature MLOps pipeline for generating trading signals from cryptocurrency OHLCV data. 
Computes rolling mean signals with comprehensive logging and metrics output.

## Features

- Reproducible signal generation with configurable random seed
- Rolling mean calculation and signal generation
- Comprehensive logging to file and console
- Structured metrics output in JSON format
- Robust error handling for various failure scenarios
- Docker containerization for deployment
